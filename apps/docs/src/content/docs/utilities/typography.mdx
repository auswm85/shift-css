## Font Size

Fluid typography that scales with viewport:

```html
<p class="text-xs">Extra small text</p>
<p class="text-sm">Small text</p>
<p class="text-base">Base text (default)</p>
<p class="text-lg">Large text</p>
<p class="text-xl">Extra large text</p>
<p class="text-2xl">2x large text</p>
<p class="text-3xl">3x large text</p>
<p class="text-4xl">4x large text</p>
<p class="text-5xl">5x large text</p>
```

## Prose

For long-form content with good defaults:

```html
<article class="prose">
  <h1>Article Title</h1>
  <p>Paragraphs, lists, and other content automatically styled.</p>
</article>
```

## CSS Custom Properties

```css
:root {
  /* Font families */
  --font-sans: 'Inter', system-ui, sans-serif;
  --font-serif: 'Georgia', serif;
  --font-mono: 'JetBrains Mono', monospace;

  /* Base size (affects all fluid typography) */
  --text-base: 1rem;
}
```
