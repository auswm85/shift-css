## Seed Hues

Override these variables to regenerate entire color palettes:

```css
:root {
  --shift-hue-primary: 250;    /* Blue (default) */
  --shift-hue-secondary: 180;  /* Cyan */
  --shift-hue-accent: 30;      /* Orange */
  --shift-hue-success: 145;    /* Green */
  --shift-hue-warning: 45;     /* Amber */
  --shift-hue-danger: 25;      /* Red */
  --shift-hue-neutral: 250;    /* Blue-tinted grays */
}
```

## Color Scales

Each color has an 11-step scale (50-950):

### Primary Colors

| Token | Lightness | Usage |
|-------|-----------|-------|
| `--color-primary-50` | 0.978 | Lightest backgrounds |
| `--color-primary-100` | 0.936 | Hover backgrounds |
| `--color-primary-200` | 0.881 | Subtle borders |
| `--color-primary-300` | 0.827 | Disabled states |
| `--color-primary-400` | 0.742 | Secondary elements |
| `--color-primary-500` | 0.648 | Primary interactive |
| `--color-primary-600` | 0.573 | Hover states |
| `--color-primary-700` | 0.469 | Active states |
| `--color-primary-800` | 0.394 | High contrast |
| `--color-primary-900` | 0.320 | Near black |
| `--color-primary-950` | 0.238 | Darkest |

### Using Color Tokens

```css
.my-element {
  background: var(--color-primary-100);
  color: var(--color-primary-900);
  border-color: var(--color-primary-300);
}
```

```html
<div style="background: var(--color-accent-500);">
  Accent background
</div>
```

## Semantic Color Tokens

Semantic tokens map to raw colors and adapt to light/dark mode:

### Surface Colors

| Token | Light Mode | Dark Mode | Usage |
|-------|------------|-----------|-------|
| `--surface-base` | neutral-50 | neutral-950 | Default backgrounds |
| `--surface-raised` | neutral-100 | neutral-900 | Cards, modals |
| `--surface-sunken` | neutral-200 | neutral-800 | Inputs, wells |

### Text Colors

| Token | Light Mode | Dark Mode | Usage |
|-------|------------|-----------|-------|
| `--text-primary` | neutral-900 | neutral-50 | Main text |
| `--text-secondary` | neutral-600 | neutral-400 | Muted text |
| `--text-disabled` | neutral-400 | neutral-600 | Disabled text |

### Border Colors

| Token | Light Mode | Dark Mode | Usage |
|-------|------------|-----------|-------|
| `--border-default` | neutral-200 | neutral-800 | Default borders |
| `--border-strong` | neutral-400 | neutral-600 | Emphasized borders |
| `--border-focus` | primary-500 | primary-400 | Focus rings |

### Interactive Colors

| Token | Light Mode | Dark Mode | Usage |
|-------|------------|-----------|-------|
| `--interactive-primary` | primary-600 | primary-400 | Buttons, links |
| `--interactive-hover` | primary-700 | primary-300 | Hover states |
| `--interactive-active` | primary-800 | primary-200 | Active states |

## Brand-Tinted Neutrals

The `--shift-hue-neutral` variable adds a subtle tint to all gray colors:

```css
/* Default: Blue-tinted grays */
--shift-hue-neutral: 250;

/* Warm grays */
--shift-hue-neutral: 30;

/* Cool grays */
--shift-hue-neutral: 210;

/* Match your primary color */
--shift-hue-neutral: var(--shift-hue-primary);
```

## Customizing Colors

### Change Brand Colors

```css
:root {
  --shift-hue-primary: 280;    /* Purple */
  --shift-hue-neutral: 280;    /* Purple-tinted grays */
}
```

### Override Specific Steps

```css
:root {
  /* Make primary-500 more saturated */
  --color-primary-500: oklch(0.65 0.2 var(--shift-hue-primary));
}
```

### Custom Semantic Mappings

```css
:root {
  --interactive-primary: var(--color-primary-700);
  --surface-base: var(--color-neutral-100);
}
```

## OKLCH Color Format

All colors use the OKLCH color space:

```css
oklch(L C H)
/* L = Lightness (0-1) */
/* C = Chroma (0-0.4) */
/* H = Hue (0-360) */
```

This ensures perceptually uniform colors across all hues.
