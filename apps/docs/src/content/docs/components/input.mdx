## Overview

Form inputs with automatic dark mode support and accessible focus states.

## Text Input

```html
<input type="text" class="input" placeholder="Enter text..." />
```

## With Labels

```html
<label class="input-group">
  <span class="input-label">Email</span>
  <input type="email" class="input" placeholder="you@example.com" />
</label>
```

## Input Sizes

```html
<input type="text" class="input input--sm" placeholder="Small" />
<input type="text" class="input" placeholder="Default" />
<input type="text" class="input input--lg" placeholder="Large" />
```

## CSS Custom Properties

| Property | Default | Description |
|----------|---------|-------------|
| `--input-bg` | `var(--surface-sunken)` | Background color |
| `--input-border` | `var(--border-default)` | Border color |
| `--input-radius` | `var(--radius-base)` | Border radius |
| `--input-padding-x` | `var(--spacing-3)` | Horizontal padding |
| `--input-padding-y` | `var(--spacing-2)` | Vertical padding |
| `--input-focus-ring` | `var(--color-primary-500)` | Focus ring color |

## Customization Examples

### Rounded Inputs

```css
.input {
  --input-radius: var(--radius-full);
}
```

### Custom Focus Color

```css
.input {
  --input-focus-ring: var(--color-accent-500);
}
```
