## Overview

Form inputs with automatic dark mode support and accessible focus states.

## Text Input

```html
<input s-input type="text" placeholder="Enter text..." />
```

## With Labels

```html
<div s-field>
  <label s-label for="email">Email</label>
  <input s-input type="email" id="email" placeholder="you@example.com" />
</div>
```

## Input Sizes

Use the `s-size` attribute:

```html
<input s-input s-size="sm" type="text" placeholder="Small" />
<input s-input type="text" placeholder="Default" />
<input s-input s-size="lg" type="text" placeholder="Large" />
```

## Form Attributes

| Attribute | Purpose | Example |
|-----------|---------|---------|
| `s-input` | Text input styling | `<input s-input>` |
| `s-field` | Form field wrapper | `<div s-field>` |
| `s-label` | Field label | `<label s-label>` |
| `s-checkbox` | Checkbox styling | `<input s-checkbox type="checkbox">` |
| `s-radio` | Radio button styling | `<input s-radio type="radio">` |
| `s-toggle` | Toggle switch | `<input s-toggle type="checkbox">` |

## How It Works

```css
:where([s-input]) {
  --_bg: var(--s-surface-sunken);
  --_border: var(--s-border-default);
  --_radius: var(--s-radius);

  background: var(--_bg);
  border: 1px solid var(--_border);
  border-radius: var(--_radius);
  padding: var(--s-space-2) var(--s-space-3);
}

[s-input]:focus {
  outline: 2px solid var(--s-focus-ring);
  outline-offset: 2px;
}
```

## Customization Examples

  --_radius: var(--s-radius-full);

### Custom Focus Color
